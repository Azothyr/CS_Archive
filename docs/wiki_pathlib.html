Pathlib Guide

pathlib provides an object-oriented interface for working with filesystem paths in Python.

Creating Paths

Use Path() to create a new path object:

from pathlib import Path
p = Path('.')

Example: This will create a path object for the current directory.

Joining Paths

You can join paths using the / operator:

p = Path('/usr')
bin_dir = p / 'bin'

Example: This will create a new path pointing to '/usr/bin'.

Navigation

Access parent directories or parts of the path:

p.parent
p.parents[0]
p.parts

Example: Navigate through the parts of the path, access the parent, or get all components of the path.

Globbing

Find files or directories matching a pattern:

for file in p.glob('*.txt'):
  print(file)

Example: Find and print all text files in the current directory.

Checking Path Properties

Check if a path exists, if it's a file, or if it's a directory:

p.exists()
p.is_file()
p.is_dir()

Example: Check if a path points to an existing location, a file, or a directory.

Working with the Home Directory

Get the user's home directory:

home = Path.home()

Example: Retrieve the path to the user's home directory.

Converting to String

If you need the string representation of the path:

str(p)

Example: Convert a path object to its string representation.

Resolve

Get the absolute path or resolve symbolic links:

p.resolve()

Example: Resolve the path to its absolute form, which can be useful when dealing with symbolic links.

Creating and Removing Directories

Create or remove directories:

p.mkdir()
p.rmdir()

Example: Create a new directory or remove an existing one.

Reading and Writing Files

Quickly read or write text from/to files:

content = p.read_text()
p.write_text('Hello, pathlib!')

Example: This reads the content of a file into a string, then writes a string to a file.

For more details, refer to the official Python documentation.
